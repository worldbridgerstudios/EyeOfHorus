# Complete Phonemic Lexicon: The Hourglass Grammar

## Framework

Each phoneme encodes:
- **1 core verb** ‚Äî the action principle
- **2 modes** ‚Äî Masculine (force/magnitude) / Feminine (relation/care)
- **3 poles per mode** ‚Äî Minima / Equilibria / Maxima
- **1 shared center** ‚Äî Equilibrium is the same in both modes

**5 positions per phoneme:** 2 masc poles + 1 shared equilibrium + 2 fem poles

## Vowel ‚Üí Mode Selection

| Sound | Mode | Effect |
|-------|------|--------|
| "ah" (a) | Masculine | Force, magnitude, external action |
| "ay/ey" (◊ê, aleph) | Feminine | Relation, care, internal quality |
| O/U | Stress | Intensity within selected mode |
